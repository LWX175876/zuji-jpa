## v1.0.1 bug 修复及小功能调整 (2018-09-01)

- **新增**了可以使用 `spring.jpa.show-sql` 的配置来作为是否打印 SQL 信息的依据之一；
- **新增**了部分类的单元测试，提高了单元测试覆盖率；
- **重构**了 XML 文件的扫描读取机制；
- **修复**了 Spring Boot web 项目打成 `jar` 包时读取不到 Fenix `XML` 文件的问题；
- **删除**了 Fenix `debug` 模式的功能；
